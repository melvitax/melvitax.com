# melvitax.com


### Installing Jekyll
```
$ gem install jekyll
```

### Installing Bundler (2.0.1 is the version supported by Netlify CMS)
```
$ gem install bundler -v 2.0.1
```

### Installing Gems
```
$ bundle install
```

## Develop with local server

```
$ bundle exec jekyll serve

```

## Compile

```
$ bundle exec jekyll build
```
